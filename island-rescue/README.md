# 🏝️ 海岛救助站

一款面向亲子用户的经营类桌面游戏，救助海洋动物，建设最美海岛！

---

## 🚀 快速开始

### 玩家版（直接玩）

1. **双击运行**：找到 `海岛救助站.exe` 文件，双击即可开始游戏
2. **无需安装**：便携版无需安装，解压后直接运行
3. **游戏窗口**：会自动打开一个独立的游戏窗口，不需要打开浏览器

### 开发者版（从源码运行）

#### 环境要求
- Node.js 18 或更高版本
- npm 或 yarn 包管理器

#### 安装依赖
```bash
cd island-rescue
npm install
```

#### 开发模式运行
```bash
npm run electron:dev
```
这会同时启动 Vite 开发服务器和 Electron 桌面窗口。

#### 构建生产版本
```bash
# 构建 Windows 桌面版（包含安装程序和便携版）
npm run electron:build:win

# 或构建所有平台版本
npm run electron:build
```

构建完成后，可执行文件会在 `release/` 目录下。

#### 直接运行已构建的版本
```bash
# 先构建前端资源
npm run build

# 然后直接启动 Electron
npm start
```

## 🎮 游戏玩法

### 核心循环
1. **🏖️ 海岸巡查**：在海滩上寻找受伤的动物和漂流物
2. **🏥 诊疗室**：给动物治疗和照护，帮助它们康复
3. **📦 仓库**：管理物资、升级仓库、装饰救助站
4. **📖 图鉴**：查看你救助过的动物和它们的康复故事
5. **📋 任务**：完成任务获得奖励，查看活动日志

### 主要功能

#### 🏛️ 救助站等级系统
- 救助动物获得声望，声望自动提升救助站等级
- 共 5 个等级：临时救助点 → 小型救助站 → 中型救助站 → 大型救助站 → 国家级保护中心
- 每级解锁新内容：更多照护位、新装饰、更高等级的动物

#### 📊 每日账本
- 在仓库「账本」标签页查看今日收支
- 记录收入、支出、净收入、获得声望
- 统计救助、放归、收集、出售数量
- 最近 7 天历史记录，经营状况一目了然

#### 💖 动物个性系统
- 每种动物都有独特的个性：胆小、贪吃、爱玩、温顺、好奇
- 每种动物都有偏好的照护方式：喂食、清洁、玩耍
- 照护方式匹配偏好时，恢复速度 +50%，健康值 +5
- 放归记录中可以看到个性和偏好匹配情况

#### 🚀 升级路线
- **💊 医疗升级**：提升治疗效果，减少物资消耗
- **💚 康复升级**：加快恢复速度，增加照护位容量
- **🔍 探索升级**：提高发现稀有动物概率，增加漂流物价值

#### 📦 待处理物品队列
- 仓库满时收集的物品不会丢失，会进入待处理列表
- 待处理物品跨天保留，永远不会消失
- 出售物品或升级仓库容量后，可以随时回来收集
- 可以选择收集或放弃待处理物品

#### 🎉 节日活动
- 随机触发节日活动，持续数天
- 节日期间有专属任务和奖励
- 节日结束后生成详细日志
- 记录具体救助/放归的动物名称、实际收支、任务完成情况

#### 🛏️ 照护互动
- 康复期间会出现喂食、清洁、玩耍等小事件
- 完成事件可以缩短恢复时间
- 匹配动物偏好的照护有额外加成
- 所有照护记录都会保存在康复故事中

## 💡 游戏技巧

1. **关注天气**：不同天气影响动物和漂流物出现概率
2. **匹配偏好**：注意动物的个性和偏好照护，匹配后恢复更快
3. **及时升级**：升级仓库容量可以收集更多物品
4. **解锁照护位**：更多照护位可以同时救助更多动物
5. **完成任务**：任务奖励丰厚，不要忘记领取
6. **参与节日**：节日活动有专属奖励，不要错过
7. **查看账本**：了解收支情况，优化经营策略

## ❓ 常见问题

**Q: 游戏数据会丢失吗？**
A: 不会，游戏数据自动保存在本地，下次打开会自动恢复。

**Q: 如何重置游戏？**
A: 按 `Ctrl+R` 重新开始，或在游戏中清除浏览器数据。

**Q: 窗口太小/太大怎么办？**
A: 使用 `Ctrl++`/`Ctrl+-` 调整缩放，或按 `F11` 全屏。

**Q: 仓库满了怎么办？**
A: 可以去仓库出售物品，或升级仓库容量。待处理物品不会丢失。
